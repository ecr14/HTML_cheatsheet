Emma's HTML Cheatsheet

Common HTML tags

Tag name End tag Description
!DOCTYPE N Defines the type of document i.e. HTML
html Y Necessary tag for all HTML content in a HTML file
head Y Holds the metadata of the file i.e. the title of the webpage, links to external fonts etc.
title Y The title of the webpage, often shown on the top of tab or browser
link N Gives a relationship between the document and an external source i.e. linking a font to the file
body Y Defines the documents body i.e. the content to display is placed here
p Y Defines a paragraph
h1..h6 Y Defines a heading with decreasing size i.e. h1 is the largest, h6 the smallest

Images, Audio & Video

Tag elements

Tag name End tag Description
img N Defines an image, use src='' to specify
audio Y Defines sound content, needs src. Use controls attribute for pause, play buttons etc.
video Y Defines a video or movie, needs src

With audio and video tags, alternative source providing can be used with the <source> tag. This goes in between the original tags
